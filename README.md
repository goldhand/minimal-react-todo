# Minimal React Todo Example

# Run this

- clone the repository
- `% npm install`
- `% npm start`

# Release this

- `% npm build`

# Steps to develop this:

1. `% npx create-react-app todo` using [create react app](https://github.com/facebook/create-react-app)
